# template telegram bot

***

### RU
Шаблон telegram бота

### Запуск проекта

1. Создать файл .env
2. Добавить переменную окружения BOT_TOKEN={`Токен бота`}
3. Запустить ```./local_build.sh```
4. Запустить ```./local_run.sh```

### Список команд

- /start - Начать диалог
- /help - Получить справку
